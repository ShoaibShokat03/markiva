## Key strategic insight

Competitors win with a NETWORK of satellite tool + reference pages (viewer, cheatsheet,
markdown-to-pdf, markdown-to-word). markdowndocs.online has the editor but none of the
high-volume reference/tool pages. Meanwhile the AI .md cluster (AGENTS.md spec by OpenAI Aug 2025,
llms.txt proposed Sept 2024) is hot, low-competition, and we already have a head start there.
Strategy: OWN the AI-docs cluster + add the evergreen reference pages (cheat sheet) to capture
top-of-funnel volume that feeds the editor/download.
