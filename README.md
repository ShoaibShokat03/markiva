# Markdown Docs

Markdown Docs is a fast Windows desktop Markdown editor and viewer built with Wails, Go, React, and Vite.

## Features

- Open `.md`, `.markdown`, `.mdown`, and `.mkd` files from the app.
- Open files through Windows "Open with" or by launching the app with a Markdown file path.
- Edit Markdown with quick toolbar actions for headings, bold, italic, links, lists, tasks, quotes, and code blocks.
- Switch between editor-only, split editor/preview, and preview-only modes.
- Save and Save As with native Windows file dialogs.
- Recent file list stored locally in the app UI.
- Light and dark themes.
